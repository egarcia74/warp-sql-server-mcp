üîí Enhanced Security Features (v1.4.0)

All MCP tools now benefit from these enterprise-grade security enhancements:

  • AST-Based Query Validation: Advanced SQL parsing with node-sql-parser to detect malicious queries and SQL injection attempts
  • Cloud Secret Management: Integrated support for AWS Secrets Manager and Azure Key Vault for secure credential storage
  • Structured Security Logging: Comprehensive audit trails for all database operations with Winston logging
  • Performance Monitoring: Real-time query performance tracking and slow query detection
  • Streaming Support: Memory-efficient handling of large result sets with configurable response formats
  • Enhanced Error Context: Detailed error reporting while protecting sensitive information

All security features are transparent to the MCP tool interface and require no changes to your existing tool calls.
